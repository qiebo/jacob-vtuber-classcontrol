# AvatarPack

运行时统一形象包目录：

- 每个子目录是一个 `pack_id`
- 结构示例：
  - `avatar_pack/<pack_id>/manifest.json`
  - `avatar_pack/<pack_id>/actions/idle/*.png`
  - `avatar_pack/<pack_id>/actions/talk/*.png`
  - `avatar_pack/<pack_id>/actions/thinking/*.png`（可选）
  - `avatar_pack/<pack_id>/actions/listening/*.png`（可选）
  - `avatar_pack/<pack_id>/actions/action/*.png`（可选）
  - `avatar_pack/<pack_id>/thumb.png`

该目录默认用于上传/转换后的用户素材，不建议手工编辑。
